# DeepTACT
DeepTACT: predicting high-resolution chromatin contacts via bootstrapping deep learning

# Requirements
- hickle
- numpy=1.13.3
- Theano=0.8.0
- keras=1.2.0
- Scikit-learn=0.18.2

# Installation
Download DeepCAT by
```shell
git clone https://github.com/liwenran/DeepCAT
```
